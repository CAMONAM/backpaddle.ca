## Setup

This site is hosted on GitHub Pages at [backpaddle.ca](https://backpaddle.ca).

## Adding Routes

1. Create a new folder under `routes/`
2. Add `index.html` (copy from existing route as template)
3. Add `route.gpx` for download
4. Update the main `index.html` with the new route card
